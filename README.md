# Mikrotik-Scripts
Routeros 自用脚本，有问题可以提issue
------------------------------------------------------------------------
netmap_ipv6
对于多线路ipv6前缀，进行线路负载，使用netmap进行前缀替换
------------------------------------------------------------------------
rule_update
对于多线路，使用rouing/rule设置routeros源进源出，使用脚本替换规则的公网地址
